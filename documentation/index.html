Elkevent Documentation

Introduction

Elkevent is a new Modern Multi-page Responsive template for Event, Conference, Workshop, Meetup, Festival, Seminar, or any other types of public gathering events. This is coming with four beautifully designed homepage variations with Retina Ready look and feel. The Elkevent has been built with latest Bootstrap 5.x and Sass.

RTL feature included in the Elkevent template so, using for Arabic and Hebrew language based website would be quick and easier.

Also, this template has Ajax based contact form submission, well commented and easily customizable source code with detailed documentation. All of these features will help to make the template uses and customization based on the ultimate event website needs.

Template Features:
  • Multipage Demos
  • Easy to Customize
  • HTML5 & CSS3
  • Clean & Simple Design
  • Fully Responsive Layout
  • Crossbrowser Compatible with Edge, IE11+, Firefox, Safari, Opera, Chrome
  • W3 Valid
  • Retina Ready

  • Well Documented

Getting Started

Unzip the archive and find out the folder elkevent and elkevent-rtl. There are all the template files in this folder.

The file and folder structure is something like this:

  1. assets/css — folder with css files.
  2. assets/fonts — folder with icon fonts files.
  3. assets/img — folder with image files.
  4. assets/js — folder with Javascript files.
  5. assets/php — folder with form-process.php files.
  6. assets/video — folder with video files.

Upload the template files to server with the help of one of the FTP-clients like FileZilla.

Responsive Media Query

We have added some utility classes with media queries for different screen sizes. All of the custom responsive css code are under the responsive.css file at the bottom showing, how many screen size we truck for media query:

Display Size: Details
max-width : 767px 
	
	/* Here is the css code for max-width : 767px: */

	/* Small Devices, Mobile, SmartPhone, Iphone ----------- */
	@media screen and (max-width: 767px) {

	}
(min-width : 768px) and (max-width : 991px) 
	
	/* Here is the css code for (min-width : 768px) and (max-width : 991px) */

	/* iPads (portrait and landscape) ----------- */
	@media only screen and (min-width : 768px) and (max-width : 991px) {

	}
(min-width : 992px) and (max-width : 1199px) 
	
	/* Here is the css code for (min-width : 992px) and (max-width : 1199px) */

	}

	/* Here is the css code for (min-width : 1650px) */

Fonts used

By default, the template loads Poppins and Oswald font from Google Web Font Services, you can change the font with the one that suits you best.

You will find the font code in the "style.css" file:
